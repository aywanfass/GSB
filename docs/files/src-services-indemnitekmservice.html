    IndemniteKmService.php

    Service de calcul des indemnités kilométriques (Tâche 6)

    PHP Version 8

    Ce service calcule le montant des indemnités kilométriques en tenant compte de la puissance fiscale du véhicule du visiteur. Il lit sa configuration depuis config/indemnites_km.php et peut être activé ou désactivé via la clé enabled.

    Lorsque le service est désactivé, PdoGsb::calculerMontantFiche() revient au tarif unitaire défini dans la table fraisforfait.

    getConfig(): array array Lit et retourne le tableau de configuration depuis config/indemnites_km.php. Retourne une config désactivée si le fichier est absent.
    isEnabled(): bool bool Indique si le barème kilométrique par puissance est activé.

    computeMontant(int $km, string $puissance, float $tarifBDD): float float Calcule le montant des frais kilométriques selon le barème. Utilise le tarif BDD en fallback si la clé de puissance est inconnue.
